# Streakify (Flutter prototype)
Proyecto prototipo que incluye:
- App Flutter modular (lib/)
- Widget integration skeleton for Android (home_widget)
- Dark / Light themes
- Simple persistence with SharedPreferences

## Cómo usar
1. Descomprimí el ZIP.
2. Abrí la carpeta en Android Studio o VS Code.
3. Ejecute `flutter pub get`.
4. Ejecutá en un emulador o dispositivo Android.
5. Para el widget, seguí las instrucciones en android/README_WIDGET.txt
